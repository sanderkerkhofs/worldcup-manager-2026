What is this project?

Worldcup Manager 2026 is a full-stack web application for managing a football World Cup knockout tournament. It allows:

  • An administrator to manage teams, players, users, and the tournament structure
  • Referees to update match statuses and record goals for their assigned matches
  • Registered fans to follow matches, standings, and top scorers
  • The system to automatically advance the bracket when a round is completed

How to run the project

Option A — Full Docker (recommended for demo)

# From the project root folder:
docker compose -f docker-compose-fullstack.yml up --build

# Wait for all containers to start, then:
# Front-end → http://localhost:8080
# Back-end API → http://localhost:3001
# API docs (Swagger) → http://localhost:3001/api-docs

Option B — Seed the database

# After starting, seed the database with test data:
docker compose exec backend npm run db:seed

# This creates 16 teams, players, all matches, and test user accounts.

💡 Presentation tip

When asked about code you wrote with AI assistance, you don't need to pretend you wrote every line yourself. Instead, focus on showing that you understand what the code does. Be able to explain: what problem does this code solve? What would happen if it wasn't there? What are the key concepts it uses? That understanding is what matters.
